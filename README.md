RevShop - Console E-commerce Application

Overview: Secure console-based e-commerce app for buyers and sellers.

Buyer Features: Register, login, browse products, add to cart, checkout, review products, make payments.

Seller Features: Register, login, manage inventory, add/update/delete products, view orders, set discounts, monitor stock.

User Management: Change password and recover forgotten password via security questions.

Product Management: Manage product details including category, price, stock, stock threshold, description, and MRP.

Order & Cart: Buyers can add/remove items, place orders, and view history; sellers get notifications.

Architecture: Modular and layered design, extendable to web/microservices.

Database: MySQL with JDBC; ERD defines tables and relationships.

Testing & Logging: JUnit for testing, Log4J for logging.

Technologies: Java, MySQL, JDBC, Git, JUnit, Log4J.
